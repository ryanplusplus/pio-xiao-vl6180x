# pio-tiny-starter-kit
PlatformIO starter kit using [tiny](https://github.com/ryanplusplus/tiny).

## Usage
### Build
```shell
make
```

### Clean
```shell
make clean
```

### Upload
```shell
make upload
```

### (Serial) Monitor
```shell
make monitor
```

### Run (Upload and Monitor)
```shell
make run
```

### Test
```shell
make test
```
